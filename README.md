- Fast Search

## How to

# Installing StickerSlack (TestFlight)
1. Install the [TestFlight](https://apps.apple.com/us/app/testflight/id899247664) app
2. Join the TestFlight using [this link](https://testflight.apple.com/join/JWTN897S)!
3. Browse for stickers and download ones you like
4. Open an iMessage chat and open StickerSlack using the + button
5. Choose a sticker!

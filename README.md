StickerSlack is an iOS App that lets you use any Slack emoji as an iMessage sticker!

## Features
- Fast Search and responsive UI
- Emoji packs
- Share packs with friends
- GIF support 

## How to use
- Search for emojis and download them 
- All downloaded emojis can be used in iMessage
- You can also create packs and share them with friends
- To use one, open an iMessage chat and tap the plus icon next to the message bar
- Find StickerSlack and tap an emoji to send! 

## Installing StickerSlack (TestFlight)
1. Install the [TestFlight](https://apps.apple.com/us/app/testflight/id899247664) app
2. Join the TestFlight using [this link](https://testflight.apple.com/join/JWTN897S)!
3. Browse for stickers and download ones you like
4. Open an iMessage chat and open StickerSlack using the + button
5. Choose a sticker!
